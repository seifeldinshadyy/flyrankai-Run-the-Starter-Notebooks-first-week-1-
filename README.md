# FlyRank ML Internship

This repository contains my work and solutions for the **FlyRank Machine Learning Internship**.

The internship focuses on applying machine learning techniques to real-world SEO and search performance data using Python, SQL, DuckDB, Pandas, and Scikit-learn.

## Completed Work

### Week 1 – Data Discovery

- Explored anonymized SEO dataset
- Investigated correlations between search metrics
- Tested hypotheses using exploratory data analysis
- Documented observations using evidence-based conclusions

---

### Week 2 – First Readable Model

- Built a rule-based baseline
- Created an interpretable Decision Tree model
- Compared Precision@20 and Precision@50
- Learned about data leakage and model evaluation

---

### Week 3 – Full Warehouse Analysis

- Connected to the FlyRank Warehouse using DuckDB
- Loaded large-scale parquet datasets directly from Hugging Face
- Built SQL-based feature engineering pipelines
- Added query-level signals
- Trained a Random Forest classifier
- Evaluated model performance using classification metrics
